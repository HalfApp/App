As the world's largest code hosting platform, GitHub’s pure English interface creates usage barriers for Chinese users. This article systematically summarizes two methods to set up a Chinese-language interface on GitHub, covering account language settings and browser plugin-based localization solutions, helping users overcome language obstacles.
Account Language Settings (Partial Effectiveness)
Chinese display can be enabled for parts of the interface through GitHub account settings, although coverage is limited. The specific steps are as follows:
Log in to GitHub and click your avatar in the upper-right corner.
Select Settings → Public profile.
In the Preferred spoken language dropdown menu, choose 中文 (Chinese).
⚠️ Note: This method only takes effect on partial interfaces such as notification emails; the main GitHub interface remains in English.
Browser Plugin Full Localization (Recommended)
Install a Script Manager
First, install a script manager such as Tampermonkey or Violentmonkey:
Tampermonkey download
Violentmonkey download
After installation, enable the extension in your browser’s extension management page.
Install the GitHub Chinese Localization Script
Search for github-chinese on GitHub and select the project maboloshi/github-chinese.
Follow the instructions in the README to install the localization script (e.g., github-chinese.user.js).
Once installed, refresh the GitHub page, and the interface will automatically switch to Chinese.
